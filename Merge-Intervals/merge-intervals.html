Merge Intervals

Treat every interval like a calendar booking. Sort all bookings by start time, then sweep left→right keeping one open block. If the next booking starts before (or exactly when) the open block ends, absorb it. Otherwise the open block is finalized and a fresh block begins.

Time  O(n log n) Space  O(n) Pattern  Sort + Linear Scan
