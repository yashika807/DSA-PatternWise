Maximum CPU Load

Same shape as Minimum Meeting Rooms, but every job carries a weight (load). Keep a min-heap of (end, load), sorted by end. Before each job, pop off every job that already finished and subtract its load. Add the new job's load, then check if this is a new peak.

Time  O(n log n) Space  O(n) Pattern  Sort + Min-Heap (weighted)
