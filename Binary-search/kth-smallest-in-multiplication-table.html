Kth Smallest Number in Multiplication Table

Same technique as Kth Smallest in Sorted Matrix — binary search over values (1 to m*n), but the table is implicit (table[i][j] = i*j), so counting uses a per-row formula min(n, x/i) instead of a staircase walk — O(m), no explicit matrix needed.

Time  O(m log(m·n)) Space  O(1) Pattern  Binary Search on the Answer (Value Range)
