Search a 2D Matrix

Rows are sorted, and each row's first element beats the previous row's last — so the whole matrix behaves like one flat sorted array. We binary search over a flattened index 0 .. m*n-1, mapping mid to row = mid / n, col = mid % n.

Time  O(log(m·n)) Space  O(1) Pattern  Binary Search — Flattened 2D Array
EXAMPLE:
Matrix (row-sorted, row-chained)
