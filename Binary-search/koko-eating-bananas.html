Koko Eating Bananas

This is NOT array-index search. We binary search over possible eating speeds (1 to max(piles)) — lo/hi/mid are candidate answers, not array positions. Each mid is tested with a feasibility check: can Koko finish in h hours at this speed? Feasible → try slower (hi = mid). Not feasible → go faster (lo = mid + 1).
