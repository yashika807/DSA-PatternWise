Upper Bound / Ceiling

Half-open interval [lo, hi). Every step we check mid: if nums[mid] >= target it's a candidate — keep it in range (hi = mid) and hunt left for something even smaller. If it's too small, rule it out entirely (lo = mid + 1). When lo == hi, that's the ceiling index — the railway timetable's next train.

Time  O(log n) Space  O(1) Pattern  Binary Search — Template B (half-open, lower bound)
