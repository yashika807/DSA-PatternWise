Rules of the game

Ek valid cycle kya hota hai?

Index i par khade ho: nums[i] positive ho to aage, negative ho to peeche jao. Array circular hai — end se aage first par, start se peeche last par.

condition 1

Indices ka sequence repeat ho: a → b → … → a

condition 2

Cycle ke saare elements ek hi sign — sab +ve ya sab -ve

condition 3

Length k > 1 — ek element ka apne aap par self-loop count nahi hota

The 3 traps

Yahi cheezein isse Medium banati hain

Trap 1 · wraparound

Movement galat ho sakta hai

Circular + negative numbers. Seedha i+nums[i] se index out-of-bounds ya negative aa jaayega. Isliye do baar modulo:

((i + nums[i]) % n + n) % n
Trap 2 · direction

Sign consistent hona chahiye

Har step par naye element ka sign purane jaisa ho. Same-sign check ek multiplication se — product > 0 matlab same sign:

nums[a] * nums[b] > 0
Trap 3 · k > 1

Self-loop reject karo

Agar next(i) == i (yaani nums[i] % n == 0), to woh k=1 ka fake loop hai. Meet hone par yeh check zaroori:

if (slow == next(slow)) break;
