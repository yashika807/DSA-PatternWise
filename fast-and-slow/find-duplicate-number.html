Find the Duplicate Number

nums[n+1] array hai jisme values 1..n range mein hain, ek duplicate ke saath. Isse modify ya extra array use nahi kar sakte. Trick: array ko khud ek **implicit linked list** samjho — index i ka "next" hai nums[i]. Duplicate value ka matlab hai do alag indices same target ko point karte hain — yani ek cycle ban jaati hai. Wahi Floyd's cycle detection se pakdo!
