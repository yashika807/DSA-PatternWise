Contiguous Array

Treat every 0 as -1 and every 1 as +1, then walk the array keeping a running balance. The ledger stores only the first index each balance was ever seen — a "photo," never overwritten. Whenever the same balance reappears, the gap since that first photo is a zero-sum stretch: equal zeros and ones. Track the longest such gap.
