3Sum Closest

Sort the range. Fix the Post (i). Send Left Arrow (left) and Right Arrow (right) toward each other. Every combo is checked against the Best Shot So Far — whichever sum lands closest to the bullseye (target) wins. Sum < target? Left Arrow steps in. Sum > target? Right Arrow steps in. Exact hit? Done instantly — nothing can be closer.

Time  O(n²) Space  O(1) extra* Pattern  Sort + Two Pointers
