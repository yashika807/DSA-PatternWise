Backspace String Compare

Two Editors read their manuscripts backward — because a # only affects the character before it. Editor S and Editor T each track an eraser count: hit #, count goes up; hit a letter while the count is positive, it's erased; otherwise it survives and gets compared. Any mismatch, or one running out before the other, means false.
